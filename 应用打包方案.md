# Pic2PPT 应用打包方案

## 1. 项目概述

**项目名称**: Pic2PPT - AI图片转PPT可编辑形状

**核心功能**:
```
PNG/JPG/WebP → AI Vision API → SVG → python-pptx → PPTX (可编辑形状)
```

**应用形态**: 绿色便携版，单EXE文件（多文件模式更稳定）

## 2. 技术架构

### 2.1 整体流程

```
┌─────────────┐     ┌─────────────────┐     ┌─────────────┐     ┌─────────────┐
│   输入图片   │────▶│   AI Vision API  │────▶│   SVG生成   │────▶│  PPTX转换   │
│ (PNG/JPG等) │     │  (Kimi/Claude)  │     │  (XML格式)  │     │ (可编辑形状) │
└─────────────┘     └─────────────────┘     └─────────────┘     └─────────────┘
```

### 2.2 核心模块

| 模块 | 路径 | 功能 |
|------|------|------|
| AI Client | `src/png2svg/ai_client.py` | 多提供商AI API客户端 |
| SVG Validator | `src/png2svg/validator.py` | SVG完整性验证 |
| Pipeline | `src/pipeline.py` | 完整转换流程 |
| SVG Parser | `src/svg_converter/parser.py` | SVG解析器 |
| Shape Handlers | `src/svg_converter/handlers/` | 各类形状处理器 |
| Converter | `src/svg_converter/converter.py` | 主转换器 |

### 2.3 技术栈

| 组件 | 选择 | 版本要求 |
|------|------|----------|
| Python | 3.7+ | 兼容旧环境 |
| GUI框架 | tkinter | Python内置 |
| PPT生成 | python-pptx | 0.6.21+ |
| SVG解析 | lxml | 4.9.0+ |
| 图片处理 | Pillow | 9.0+ |
| AI SDK | openai | 兼容多提供商 |
| 打包工具 | PyInstaller | 6.0+ |

## 3. PyInstaller 打包配置

### 3.1 配置说明

当前使用**单文件模式**（更便携），生成独立EXE：
- `pic2ppt.exe` - 主程序（33MB，包含所有依赖）

**打包命令**:
```bash
pyinstaller pic2ppt_onefile.spec --clean --noconfirm
```

### 3.2 打包脚本

**单文件模式** (`pic2ppt_onefile.spec`):
```python
# -*- mode: python ; coding: utf-8 -*-
from PyInstaller.utils.hooks import collect_all

# 收集所有依赖数据
anthropic_datas = collect_all('anthropic')
openai_datas = collect_all('openai')
httpx_datas = collect_all('httpx')

a = Analysis(
    ['pic2ppt.py'],
    pathex=[str(BASE_DIR)],
    datas=[
        ('src', 'src'),  # 内嵌src目录
        ('tkinterdnd2/tkdnd', 'tkdnd'),  # 拖拽支持
    ] + anthropic_datas[0] + openai_datas[0] + httpx_datas[0],
    hiddenimports=[
        'anthropic', 'anthropic.types', 'anthropic._client',
        'openai', 'openai.types', 'openai.types.chat', 'openai._client',
        'httpx', 'jiter',
        'pptx', 'pptx.util', 'pptx.dml', 'pptx.enum', 'pptx.oxml',
        'pptx.api', 'pptx.package', 'pptx.opc', 'pptx.parts',
        'pptx.shapes', 'pptx.text', 'pptx.dml.color',
        'lxml', 'lxml.etree', 'lxml._elementpath',
        'PIL', 'PIL.Image', 'PIL.ImageTk', 'PIL.ImageGrab',
        'src.pipeline', 'src.png2svg', 'src.png2svg.ai_client',
        'src.svg_converter', 'src.svg_converter.handlers',
    ],
    excludes=[  # 排除以减少体积
        'matplotlib', 'numpy', 'pandas', 'scipy', 'sklearn',
        'tensorflow', 'torch', 'pytest', 'unittest', 'pydoc',
        'doctest', 'ctypes.test',
        'distutils', 'setuptools', 'pip', 'wheel', 'pkg_resources',
        '_pytest', 'test', 'tests', 'IPython', 'jupyter', 'notebook',
        'PyQt5', 'PyQt6', 'PySide2', 'PySide6', 'wx', 'wxPython',
    ],
    # ...
)

# 单文件模式配置
exe = EXE(
    pyz,
    a.scripts,
    a.binaries,
    a.zipfiles,
    a.datas,
    name='pic2ppt',
    console=False,  # GUI模式，无控制台窗口
    # ...
)
```

**多文件模式** (`pic2ppt.spec`) - 备选方案：
```python
# 多文件模式：生成目录结构（启动更快）
exe = EXE(
    pyz,
    a.scripts,
    [],
    exclude_binaries=True,  # 关键：排除二进制文件到单独目录
    name='pic2ppt',
    console=False,
)

coll = COLLECT(
    exe,
    a.binaries,
    a.zipfiles,
    a.datas,
    name='pic2ppt',
)
```

## 6. 代码修复记录（已完成）

以下修复已全部实现并验证：

### 6.1 修复1: RectHandler检测圆形 ✅

**实现**: `src/svg_converter/handlers/basic_shapes.py`

当检测到rx ≈ width/2且ry ≈ height/2时，自动使用OVAL形状：

```python
if rx > 0 or ry > 0:
    # 检测是否为圆形/椭圆
    is_circle_like = (abs(rx - width/2) < 2 and abs(ry - height/2) < 2)
    if is_circle_like:
        shape = context.slide.shapes.add_shape(MSO_SHAPE.OVAL, left, top, w, h)
    else:
        shape = context.slide.shapes.add_shape(MSO_SHAPE.ROUNDED_RECTANGLE, left, top, w, h)
```

### 6.2 修复2: PathHandler曲线稳定性 ✅

**实现**: `src/svg_converter/handlers/polygons.py`

增强圆弧计算的数值稳定性：

```python
if rx == 0 or ry == 0:
    return [start, end]

# 增加数值保护
try:
    lambda_val = (x1p ** 2) / (rx ** 2) + (y1p ** 2) / (ry ** 2)
    if lambda_val > 1:
        sqrt_lambda = math.sqrt(lambda_val)
        rx *= sqrt_lambda
        ry *= sqrt_lambda
except (ZeroDivisionError, ValueError):
    return [start, end]
```

### 6.3 修复3: Polyline折线保持 ✅

**实现**: `src/svg_converter/handlers/polygons.py`

多点折线使用多段连接器拼接，保持视觉连续性：

```python
for i in range(len(points) - 1):
    x1, y1 = points[i]
    x2, y2 = points[i + 1]
    connector = context.slide.shapes.add_connector(
        MSO_CONNECTOR.STRAIGHT, begin_x, begin_y, end_x, end_y
    )
    self._apply_line_style(connector, style)
```

### 6.4 修复4: 文本位置优化 ✅

**实现**: `src/svg_converter/handlers/text.py`

根据字体类型调整基线偏移：

```python
font_family = style.font_family or 'Microsoft YaHei'
if 'YaHei' in font_family or 'SimHei' in font_family:
    baseline_ratio = 0.80  # 中文字体
else:
    baseline_ratio = 0.72  # 英文字体

baseline_offset = (font_size_pt * baseline_ratio) / 72
```
