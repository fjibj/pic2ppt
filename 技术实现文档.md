# AI图片转PPT可编辑形状 - 技术实现文档

## 1. 项目概述

本项目实现了AI驱动的图片到PPT可编辑形状的自动化转换流程：

```
PNG/JPG/WebP → AI Vision API → SVG → python-pptx → PPTX (可编辑形状)
```

## 2. 架构设计

### 2.1 整体流程

```
┌─────────────┐     ┌─────────────────┐     ┌─────────────┐     ┌─────────────┐
│   输入图片   │────▶│   AI Vision API  │────▶│   SVG生成   │────▶│  PPTX转换   │
│ (PNG/JPG等) │     │  (Kimi/Claude)  │     │  (XML格式)  │     │ (可编辑形状) │
└─────────────┘     └─────────────────┘     └─────────────┘     └─────────────┘
```

### 2.2 核心模块

- **ai_client.py**: AI API客户端，支持多提供商
- **parser.py**: SVG解析器
- **handlers/**: 元素处理器集合
- **converter.py**: 主转换器

## 3. 关键技术问题与解决方案

### 3.3 SVG到PPT转换问题

#### 3.3.1 形状类型映射

**问题**: SVG使用`<rect rx="35">`表示圆形，PPT识别为圆角矩形

**解决方案**: 在SVG中使用真正的`<circle>`元素：

```xml
<!-- 不推荐 -->
<rect x="380" y="70" width="70" height="70" rx="35" fill="#7c3aed"/>

<!-- 推荐 -->
<circle cx="415" cy="105" r="35" fill="#7c3aed"/>
```

**处理器映射表**:

| SVG元素 | PPT形状 | 处理器 |
|---------|---------|--------|
| `<rect>` | MSO_SHAPE.RECTANGLE | RectHandler |
| `<rect rx="5">` | MSO_SHAPE.ROUNDED_RECTANGLE | RectHandler |
| `<circle>` | MSO_SHAPE.OVAL | CircleHandler |
| `<ellipse>` | MSO_SHAPE.OVAL | EllipseHandler |
| `<line>` | MSO_CONNECTOR.STRAIGHT | LineHandler |
| `<polygon>` | 自动识别 | PolygonHandler |
| `<path>` | 分析后转换 | PathHandler |

#### 3.3.2 曲线转换问题

**问题**: SVG中的曲线路径(Q/C/A命令)在PPT中显示为折线或消失

**解决方案**: 使用线段逼近算法：

```python
def _subdivide_quadratic_bezier(self, p0, p1, p2, steps):
    """细分二次贝塞尔曲线为直线段"""
    points = []
    for i in range(steps + 1):
        t = i / steps
        # B(t) = (1-t)^2 * P0 + 2(1-t)t * P1 + t^2 * P2
        x = (1 - t) ** 2 * p0[0] + 2 * (1 - t) * t * p1[0] + t ** 2 * p2[0]
        y = (1 - t) ** 2 * p0[1] + 2 * (1 - t) * t * p1[1] + t ** 2 * p2[1]
        points.append((x, y))
    return points
```

**经验**: 增加steps(20-25)可获得更平滑的曲线效果。

#### 3.3.3 箭头样式问题

**问题**: SVG marker定义的箭头在PPT中不显示

**解决方案**: 使用python-pptx的XML操作添加箭头样式：

```python
def _add_arrow_to_connector(self, connector, end_type: str, line_width):
    from lxml import etree
    P_NS = 'http://schemas.openxmlformats.org/presentationml/2006/main'
    A_NS = 'http://schemas.openxmlformats.org/drawingml/2006/main'

    spPr = connector.element.find('{%s}spPr' % P_NS)
    if spPr is not None:
        ln = spPr.find('{%s}ln' % A_NS)
        if ln is not None:
            # 根据线宽选择箭头大小
            if line_width >= Pt(3):
                arrow_size = 'lg'   # 大号箭头
            elif line_width >= Pt(1.5):
                arrow_size = 'med'  # 中号箭头
            else:
                arrow_size = 'sm'   # 小号箭头

            # PPT中：headEnd = 起点箭头, tailEnd = 终点箭头（与直觉相反！）
            if end_type == 'tail':  # 终点箭头
                tailEnd = etree.SubElement(ln, '{%s}tailEnd' % A_NS)
                tailEnd.set('type', 'arrow')
                tailEnd.set('w', arrow_size)
                tailEnd.set('len', arrow_size)
            elif end_type == 'head':  # 起点箭头
                headEnd = etree.SubElement(ln, '{%s}headEnd' % A_NS)
                headEnd.set('type', 'arrow')
                headEnd.set('w', arrow_size)
                headEnd.set('len', arrow_size)
```

**关键发现**:
- PPT XML中`headEnd`和`tailEnd`的定义与直觉相反
- `headEnd` = 起点（Line Start）箭头
- `tailEnd` = 终点（Line End）箭头

**箭头大小选择**:
- 细线条（<1.5pt）：使用'sm'小号箭头
- 中线（1.5-3pt）：使用'med'中号箭头
- 粗线（≥3pt）：使用'lg'大号箭头
- 大小不匹配会导致箭头显示异常（如方块）

**优势**: 箭头成为线条的一部分，不是独立对象。

#### 3.3.5 形状填充与默认颜色

**问题**: SVG中无填充属性的形状在PPT中显示为黑色或透明

**解决方案**: 在BaseHandler中实现智能默认填充逻辑：

```python
def _apply_style(self, shape, element: SVGElement, context: RenderContext):
    style = context.parent_style.merge(element.style)

    # 填充处理
    if style.fill and style.fill.lower() == 'none':
        shape.fill.background()  # 明确无填充
    elif style.fill:
        fill_color = ColorParser.parse(style.fill)
        if fill_color:
            shape.fill.solid()
            shape.fill.fore_color.rgb = fill_color
    else:
        # 默认填充：rect/circle/ellipse/polygon默认白色
        if element.tag in ('rect', 'circle', 'ellipse', 'polygon'):
            shape.fill.solid()
            shape.fill.fore_color.rgb = RGBColor(255, 255, 255)
        else:
            shape.fill.background()
```

**关键点**:
- 显式`fill="none"`才表示无填充
- 无填充属性时，几何形状默认白色填充
- 这与PPT的默认行为不同，避免黑色矩形问题

**问题2**: 坐标格式多样化

**SVG支持**: `x,y x,y` / `x y x y` / `x,y,x,y`

**解析方案**:
```python
def parse_points(self, points_str: str) -> List[Tuple[float, float]]:
    points = []
    # 优先尝试 "x,y x,y" 格式
    coords = re.findall(r'(-?\d+\.?\d*)[\s,]+(-?\d+\.?\d*)', points_str)
    for x, y in coords:
        points.append((float(x), float(y)))

    # 如果没解析到，尝试 "x y x y" 格式
    if not points:
        numbers = re.findall(r'-?\d+\.?\d*', points_str)
        for i in range(0, len(numbers) - 1, 2):
            points.append((float(numbers[i]), float(numbers[i+1])))

    return points
```

**问题**: 旋转文本框实现竖排会导致文字倾斜

**正确方案**: 使用东亚竖排属性`eaVert`：

### 3.4 图片压缩处理

**问题**: AI API通常有文件大小限制（如15MB-20MB）

**解决方案**: 自动压缩策略：

```python
def _encode_image(self, image_path: Path) -> tuple:
    """编码图片，必要时压缩"""
    import io
    from PIL import Image

    # 检查文件大小
    file_size = image_path.stat().st_size
    max_size = 15 * 1024 * 1024  # 15MB

    if file_size > max_size:
        # 需要压缩
        with Image.open(image_path) as img:
            # 转换为RGB（去除透明通道）
            if img.mode in ('RGBA', 'P'):
                img = img.convert('RGB')

            # 计算缩放比例
            scale = math.sqrt(max_size / file_size) * 0.9
            new_size = (int(img.width * scale), int(img.height * scale))
            img = img.resize(new_size, Image.Resampling.LANCZOS)

            # 保存为JPEG，质量85%
            buffer = io.BytesIO()
            img.save(buffer, format='JPEG', quality=85, optimize=True)

            # 如果还是太大，进一步压缩到60%
            if buffer.tell() > max_size:
                buffer.seek(0)
                buffer.truncate()
                img.save(buffer, format='JPEG', quality=60, optimize=True)

            return 'image/jpeg', buffer.getvalue()

    # 不需要压缩，直接读取
    with open(image_path, 'rb') as f:
        return 'image/png', f.read()
```

**压缩策略**:
1. 首先尝试等比缩放（保持宽高比）
2. 转换为JPEG格式（通常比PNG小）
3. 质量85%优先，如仍超限制则60%
4. 保持原始图片的清晰度和可读性

### 3.5 AI响应解析增强

**问题**: Kimi API返回ThinkingBlock类型导致解析失败

**解决方案**: 遍历所有content blocks寻找text：

```python
def _call_anthropic_sdk(self, client, base_url, model, system_prompt, user_content):
    """调用Anthropic SDK（兼容Kimi）"""
    response = client.messages.create(
        model=model,
        max_tokens=8192,
        system=system_prompt,
        messages=[{"role": "user", "content": user_content}],
        timeout=300
    )

    # 遍历content blocks，兼容TextBlock和ThinkingBlock
    text_content = ""
    for block in response.content:
        if hasattr(block, 'text'):
            text_content += block.text
        elif hasattr(block, 'thinking'):
            # 处理思考块（如有需要）
            pass

    return text_content
```

**关键发现**:
- Kimi有时会返回`ThinkingBlock`类型的content
- 需要遍历所有blocks而不是直接取`content[0].text`
- Anthropic SDK通过OpenAI兼容模式使用时也要注意此问题

```python
def _draw_rounded_rect(self, canvas, x1, y1, x2, y2, radius, fill='', outline='', width=1):
    """绘制圆角矩形"""
    # 四个角的圆弧
    canvas.create_arc(x1, y1, x1 + 2*radius, y1 + 2*radius,
                      start=90, extent=90, style='pieslice', fill=fill)
    canvas.create_arc(x2 - 2*radius, y1, x2, y1 + 2*radius,
                      start=0, extent=90, style='pieslice', fill=fill)
    canvas.create_arc(x2 - 2*radius, y2 - 2*radius, x2, y2,
                      start=270, extent=90, style='pieslice', fill=fill)
    canvas.create_arc(x1, y2 - 2*radius, x1 + 2*radius, y2,
                      start=180, extent=90, style='pieslice', fill=fill)
    # 填充中间区域
    canvas.create_rectangle(x1 + radius, y1, x2 - radius, y2, fill=fill)
    canvas.create_rectangle(x1, y1 + radius, x1 + radius, y2 - radius, fill=fill)
    canvas.create_rectangle(x2 - radius, y1 + radius, x2, y2 - radius, fill=fill)
    canvas.create_rectangle(x1 + radius, y1 + radius, x2 - radius, y2 - radius, fill=fill)
```

```python
from PIL import ImageGrab

def paste_from_clipboard():
    """从剪贴板粘贴图片"""
    try:
        win32clipboard.OpenClipboard()
        if win32clipboard.IsClipboardFormatAvailable(win32clipboard.CF_BITMAP):
            # 获取位图
            img = ImageGrab.grabclipboard()
            if img:
                return img
    finally:
        win32clipboard.CloseClipboard()
    return None
```

```python
#!/usr/bin/env python3
"""pic2ppt - AI图片转PPT可编辑形状"""

import tkinter as tk
from tkinter import ttk, filedialog, messagebox
from pathlib import Path
import json
import logging

# 配置日志（保存到同级目录）
log_file = Path(__file__).parent / "pic2ppt.log"
logging.basicConfig(
    filename=log_file,
    level=logging.INFO,
    format='%(asctime)s - %(levelname)s - %(message)s'

class Pic2PPTApp:
    def __init__(self, root):
        self.root = root
        self.root.title("pic2ppt v1.0")
        self.root.geometry("600x700")
        self.root.resizable(False, False)

        self.current_image = None
        self.config = self._load_config()

        self._setup_ui()
        self._bind_events()

    def _setup_ui(self):
        """设置界面"""
        # 图片预览区
        self.preview_frame = tk.Frame(self.root, width=560, height=300, bg="#f0f0f0")
        self.preview_frame.pack(padx=20, pady=10)
        self.preview_frame.pack_propagate(False)

        self.preview_label = tk.Label(
            self.preview_frame,
            text="点击选择图片 或 拖拽 或 Ctrl+V粘贴",
            bg="#f0f0f0"
        )
        self.preview_label.pack(expand=True)

        # API配置区（可折叠）
        self._setup_api_section()

        # 输出设置
        self._setup_output_section()

        # 进度条
        self.progress = ttk.Progressbar(self.root, length=560, mode='determinate')
        self.progress.pack(padx=20, pady=10)

        self.status_label = tk.Label(self.root, text="就绪")
        self.status_label.pack()

        # 按钮
        btn_frame = tk.Frame(self.root)
        btn_frame.pack(pady=10)

        self.convert_btn = tk.Button(
            btn_frame, text="开始转换",
            command=self._on_convert,
            width=15, height=2
        )
        self.convert_btn.pack(side=tk.LEFT, padx=10)

        self.cancel_btn = tk.Button(
            btn_frame, text="取消",
            command=self._on_cancel,
            width=15, state=tk.DISABLED
        )
        self.cancel_btn.pack(side=tk.LEFT, padx=10)

    def _bind_events(self):
        """绑定事件"""
        # 拖拽
        self.preview_frame.bind('<Drop>', self._on_drop)

        # 粘贴
        self.root.bind('<Control-v>', self._on_paste)
        self.root.bind('<Control-V>', self._on_paste)

        # 点击
        self.preview_frame.bind('<Button-1>', self._on_browse)

    def _on_browse(self, event=None):
        """选择文件"""
        path = filedialog.askopenfilename(
            filetypes=[
                ("图片", "*.png *.jpg *.jpeg *.webp *.gif *.bmp"),
                ("所有文件", "*.*")
            ]
        )
        if path:
            self._load_image(path)

    def _on_drop(self, event):
        """拖拽上传"""
        files = event.data.split()
        if files:
            self._load_image(files[0])

    def _on_paste(self, event=None):
        """粘贴上传"""
        img = self._get_clipboard_image()
        if img:
            # 保存临时文件
            temp_path = Path(tempfile.gettempdir()) / "pic2ppt_paste.png"
            img.save(temp_path)
            self._load_image(str(temp_path))
        else:
            messagebox.showinfo("提示", "剪贴板中没有图片")

    def _get_clipboard_image(self):
        """获取剪贴板图片"""
        try:
            from PIL import ImageGrab
            img = ImageGrab.grabclipboard()
            if isinstance(img, Image.Image):
                return img
        except Exception as e:
            logging.error(f"获取剪贴板失败: {e}")
        return None

    def _load_config(self):
        """加载配置"""
        config_file = Path(__file__).parent / "config.json"
        if config_file.exists():
            with open(config_file, 'r', encoding='utf-8') as f:
                return json.load(f)
        return {
            "base_url": "",
            "api_key": "",
            "model": "",
            "output_dir": str(Path.home() / "Documents")
        }

    def _save_config(self):
        """保存配置"""
        config_file = Path(__file__).parent / "config.json"
        with open(config_file, 'w', encoding='utf-8') as f:
            json.dump(self.config, f, ensure_ascii=False)

if __name__ == "__main__":
    root = tk.Tk()
    app = Pic2PPTApp(root)
    root.mainloop()
```
